推荐些提升效率的工具

发表于2021-03-03|更新于2021-03-03|工具
|字数总计:2.4k|阅读时长:7分钟

Stay true to yourself if nobody else.

前言

好久没有更新，因为最近为工作的事情忙碌，想更新都没得抽空搞一搞。难得更新一次，分享一些我日常工作和生活中使用的一些能提高工作效率的工具软件！

文本编辑

作为程序员，这年头应该不会还有人用 windows 自带的记事本看代码吧，不会吧不会吧。

UltraEdit

UltraEdit 是一套功能强大的文本编辑器，可以编辑文本、十六进制、ASCII 码，完全可以取代 windows 记事本，内建英文单字检查、C++ 及 VB 指令突显，可同时编辑多个文件，而且即使开启很大的文件速度也不会慢。

UltraEdit 作为一款老牌的编辑器，不仅界面好看，而且其十六进制编辑功能强大，代码补全、折叠、高亮等功能一应俱全，还集成了 HTML / Markdown 实时预览、FTP，SSH 等功能，更多的实用功能，需要大家自己慢慢发掘。

UltraEdit 虽然功能全面且强大，但是作为一款商业编辑器，它是收费的，并且功能性齐全也导致了安装体积稍为臃肿，启动时较为缓慢，不过，作为一款编辑器，功能上应有尽有，对我嵌入式工作中，使用十六进制功能体验还是十分好的，所以我也会向周边人推荐这款编辑器。

有兴趣的小伙伴可以在我的公众号回复 10001 获取相应稳定破解版本，当然有能力的话，当然是支持正版啦。

Notepad++(不推荐使用)

Notepad++ 是一个免费的开放源代码文本和源代码编辑器。它采用 C++ 编写，该软件最大的优势在于性能优秀、小巧（完整安装包仅 3.8MB）、功能众多、插件丰富，而且最为关键的是“完全免费”！

作为之前经常使用的文本编辑器，一直是我的首推，但是，前段时间爆出 Notepad++ 作者在其官网公开发布反华言论，抹黑祖国，所以我对这款文本编辑器的好感度明显下降了，即使这款编辑器轻量且强大，但是作为国人还是不能认可该类行为，所以在我的新笔记本上也没有再装过这款软件了。

VSCODE

VSCODE 全称 Visual Studio Code，是由微软开发用于编写现代 Web 和云应用跨平台编辑器，支持几乎所有主流的开发语言的语法高亮、智能代码补全、自定义热键，括号匹配、代码片段、代码对比 Diff、GIT 等特性，支持插件扩展，并针对网页开发和云端应用开发做了优化。软件跨平台支持 Win、Mac 以及 Linux。

VSCODE 和 Notepad++在启动上不相上下，基本秒开，并且 VSCODE 支持 MARKDOWN 语法，并且支持 Git，我写博客文章的实时预览和提交都在 VSCODE 中完成的，因为 VSCODE 的体验效果十分好，所以我也经常在别的文章中提及该编辑器。

VSCODE 内置了 Git 版本控制，可以方便提交、回滚、推送代码，我现在的代码工程管理基本上都使用了 Git 和 SVN，后面可能会写一篇相关的文章来教大家，怎样的管理自己的代码，方便回溯整个项目的时间线。
VSCODE 还支持插件拓展，里面有许多实用且有趣的插件，话不多说，大家自行发掘。

VSCODE 是完全免费的编辑器，各位可以在官网上下载它。

编辑器就到这里了，其他优秀的编辑器还有很多，选择合适自己的去使用就可以了，上面三个都是我使用频率最高的编辑器了，首推 VSCODE，其次 UltraEdit，Notepad++ 就算了吧。

截图工具

Snipaste

Snipaste 是一个简单但强大的截图工具，也可以让你将截图贴回到屏幕上！下载并打开 Snipaste，按下 F1 来开始截图，再按 F3，截图就在桌面置顶显示了。就这么简单！

你还可以将剪贴板里的文字或者颜色信息转化为图片窗口，并且将它们进行缩放、旋转、翻转、设为半透明，甚至让鼠标能穿透它们！如果你是程序员、设计师，或者是大部分工作时间都在电脑前，贴图功能将改变你的工作方式、提升工作效率。

Snipaste 是免费软件，它也很安全，没有广告、不会扫描你的硬盘、更不会上传用户数据，它只做它应该做的事。

上面是 Snipaste 官网的介绍，这个截图工具我最喜欢的就是贴图了，将截图的图片贴在电脑上，非常适合没有多个显示器时看代码的烦恼。而且该截图软件还有取色，标注等功能。下面放两张随意演示的 demo：

更多使用的细节，可以参考官网的用户手册，Snipaste 对我来说唯一的缺点就是不能滚动截屏，其他功能在使用上已经算是应有尽有了。软件的下载可以访问上面的官网，Windows 商店中也有该软件。

FastStone Capture

FastStone Capture 是一个功能强大，轻巧但功能齐全的屏幕捕获工具和屏幕录像机。它使您可以轻松捕获和注释屏幕上的所有内容，包括窗口，对象，菜单，全屏，矩形/手绘/固定区域以及滚动窗口/网页。它还允许您记录所有屏幕活动，包括屏幕上的变化，麦克风的语音，扬声器的音频，鼠标移动以及单击到高度压缩的视频文件中。您可以选择将捕获的内容发送到编辑器，文件，剪贴板，打印机，电子邮件，OneNote / Word / Excel / PowerPoint文档或将其上传到您的网站。编辑工具包括注释（文本，箭头线，高光），调整大小，裁剪，锐化，加水印，应用边缘效果等。其他功能包括图像扫描，全局热键，自动文件名生成，对外部编辑器的支持，颜色选择器。

FastStone Capture将图像保存为BMP，GIF，JPEG，PCX，PNG，TGA，TIFF和PDF格式。内置屏幕录像机以WMV（Windows Media Video）格式保存视频。

上面同样是 FastStone Capture 官网上的介绍，这个工具功能同样很多，但是由于有了 Snipaste，所以 FastStone Capture 对我来说只有在需要用到滚动截屏场景时才会使用的工具，所以使用频率少了很多，但是其内置的功能还是十分强大的。

对比工具

Beyond Compare

Beyond Compare 是一套由 Scooter Software 推出的文件比较工具。主要用途是对比两个文件夹或者文件，并将差异以颜色标示。比较范围包括目录，文档、表格、图片、注册表、十六进制等多样化的选择，可以满足不同人士不同行业的要求。

Beyond Compare 专注于智能比较，所以它比较内容的速度十分快速，比较完会高亮出文件之间的差异，并且可以选择比较形式，例如 CRC 比较、二进制比较等形式。

不过 Beyond Compare 是需要收费的，但是作为老牌的对比工具，我觉得它的功能十分强大，无论是在对比代码还是对比工程，都可以一目了然的看到差异，可以快速帮我们定位差异，是个十分不错的工具！

有兴趣的小伙伴可以在我的公众号回复 10002 获取相应稳定破解版本，有能力的话，可以支持正版。

思维导图

XMind

XMind 是一款全球领先的思维导图软件，用户量非常巨大。我们可以利用它绘制思维导图，还能绘制鱼骨图、二维图、树形图、逻辑图、组织结构图等。使用 XMind 提供各种结构图可以帮助你更好地理清复杂的想法和事项。多种视觉化的思维呈现方式让你能进行深度的对比分析，更直观地进行里程碑的记录和事项的安排。

对我来说，使用 XMind，无论是学习和工作上，都是一个不错选择的，它可以帮我更拆分出项目的框架和结构，让我更清晰的看出项目的不足和未完善的地方，对我来说也是个使用频率很高的工具之一。

不过 XMind 一直都是收费软件，还挺贵的。有兴趣的小伙伴可以在我的公众号回复 10003 获取相应稳定破解版本，有能力的话，可以支持正版。

QuickLook

QuickLook 它实现了用户只需按一下 Space 键，就可以以闪电般的速度窥视文件内容。通过 QuickLook 可以快速预览图片、音视频、压缩文件、文档、表格等。

通过 Space 键，就可以提前预览文件内容，在考虑是否需要通过相应的工具打开，这样也可以大大的节省时间。

QuickLook 的下载可以在 Github 上面获取，也可以在 Windows 商店下载。
